# Aptogotchi

## Publish Modules

```shell
./sh_scripts/move_publish.sh
```

## Run unit tests

```shell
./sh_scripts/move_test.sh
```

## Run move scripts

Create gotchi.
```shell
./sh_scripts/move_run_script_create_gotchi.sh
```

Feed gotchi.
```shell
./sh_scripts/move_run_script_feed_gotchi.sh
```
